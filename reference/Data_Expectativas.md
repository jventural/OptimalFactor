# Expectativas Dataset

A dataset containing responses to an expectations scale. This dataset is
provided as an example for demonstrating the EFA optimization functions
in the OptimalFactor package.

## Usage

``` r
data(Data_Expectativas)
```

## Format

A data frame with observations and item responses for the expectations
scale.

## Source

Simulated data for demonstration purposes.
