# Export CFA Boosting results to structured data frames

Export CFA Boosting results to structured data frames

## Usage

``` r
export_cfa_boosting(result)
```

## Arguments

- result:

  Output from
  [`cfa_boosting()`](https://jventural.github.io/OptimalFactor/reference/cfa_boosting.md)

## Value

A named list of data frames
